# 内置异常

  • 所有异常派生自BaseException
  • 自定义异常推荐从Exception类或者它的子类来派生

# 异常上下文

raise new_exc from original_exc

# 异常组

  • exception ExceptionGroup(msg, excs)
  • exception BaseExceptionGroup(msg, excs)
  • message
  • exceptions
  • subgroup
  • split(condition)
  • derive(excs)
